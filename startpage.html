BadWolf Browser for Bodhi Linux

⌨ Default Keybindings ⌨

Esc - Stops loading the content in the current tab or escapes search
F1 - Shows the about dialog
F5 - Reloads the content in the current tab without clearing cache
F7 - Toggles caret browsing
F12 - Opens the web inspector
Alt-i - Enables/disables Image loading for the current tab (requires refresh)
Alt-j - Enables/disables Javascript for the current tab (requires refresh)
Alt-No. Where No. is any numeric-row key. Go to the n-th tab, 0 goes to the last one
Alt-Left/Right - Go to the previous/next tab
Ctrl-Scroll - Zooms the webpage in/out
Ctrl-0 - Resets webpage zoom to 100%
Ctrl-t - Creates a new tab
Ctrl-F4 - Closes the current tab
Ctrl-f - Focuses on the search entry (search inside page)
Ctrl-g/G - When search box is focused it goes to the Next/Previous search term
Ctrl-l - Focuses on the location (URL) entry
Ctrl-[/] - Go back/forward in current tab's history
Ctrl-p - Print the current page (spawns a dialog)
Ctrl-PgDn/PgUp - Go to the previous/next tab
Ctrl-r/R - Reloads the content in the current tab (with/without clearing cache)
